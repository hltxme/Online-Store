💙 Alipay Face-to-Face Payment

📖 Payment Guide

To use Alipay Face-to-Face payment:

  1. Register an Alipay developer account at open.alipay.com
  2. Create an application and enable "当面付" (Face-to-Face Payment)
  3. Generate RSA2 key pair and configure in the application
  4. Copy App ID and Private Key to the fields on the left
  5. Set the payment notification URL to receive payment callbacks
Note: After a customer places an order, a QR code will be generated for them to scan with Alipay. The payment status will be updated automatically via callback.
